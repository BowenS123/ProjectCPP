# How to start up ClankerFactory

## Prerequisites
- Qt Creator (with CMake and a C++ compiler installed)
- Git (optional) if you prefer cloning instead of downloading the ZIP.

---

## Get the Project
1. **Option A – Git clone**
   - `git clone https://github.com/<your-account>/ProjectCPP.git`
   - Project root: `ProjectCPP/` (Qt sources are in `ClankerFactory/`).
2. **Option B – Download ZIP**
   - On GitHub, click **Code > Download ZIP**.
   - Unzip to a convenient folder, e.g., `C:/Projects/ClankerFactory`.

---

## Open in Qt Creator
1. Start Qt Creator.

![QtCreator](assets/QtCreator.png)

2. Go to **Select > Open Project..**

![QtCreatorHomePAge](assets/QtCreatorHomePAge.png)

3. Select `ProjectCPP/ClankerFactory/CMakeLists.txt` and click **Open**.

![FileSelect](assets/FileSelect.png)

4. If Qt Creator ask you to Configure Project, Click **Configure Project**

![ConfigureProject](assets/ConfigureProject.png)

5. Click **Run** The green button (bottom-left, under Debug).
6. The project will build and start automatically
7. You can now play the game. Good luck and have fun!

![Play](assets/Play.png)
